Harbor Light Community Center Website

Overview

This project is a simple HTML5 website created for the Harbor Light Community Center.
It demonstrates the use of semantic HTML elements, basic CSS styling, accessibility features, and structured content for a community-focused website.

The page includes information about the center, its services, upcoming events, and contact details.

---

Features

- Semantic HTML5 structure ("header", "nav", "main", "section", "footer")
- Internal CSS styling
- Accessible navigation with anchor links
- Email contact link using "mailto"
- Structured event data using Schema.org microdata
- Accessible images with proper "alt" attributes
- Decorative image hidden from screen readers using "aria-hidden"

---

Technologies Used

- HTML5
- CSS3
- Schema.org Microdata
- Basic Accessibility (ARIA attributes)

Sections of the Website

How to Run the Project

git clone https://github.com/yourusername/harbor-light-community-center.git

2. Open the project folder.

3. Double-click Harbourlights.html
   or open it in any web browser.

No server setup is required because this is a static website.
